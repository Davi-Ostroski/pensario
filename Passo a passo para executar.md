## Como Executar

### 1. Backend Node.js
```bash
cd pensario-fullstack-nodejs
npm install
npm start
# Servidor rodando em http://localhost:3000
```

### 2. Frontend React.js
```bash
cd pensario-fullstack-nodejs/pensario-react-web
pnpm install
pnpm run dev
# Aplicação web em http://localhost:5173
```

### 3. Mobile React Native
```bash
cd pensario-fullstack-nodejs/PensarioMobile
npm install

# Para Android
npx react-native run-android
